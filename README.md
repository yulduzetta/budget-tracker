## Installation
- To view and deploy the app from remote server, simply navigate to: https://yulduzetta-budget-tracker.herokuapp.com/

- To run app locally,
```typescript
$ git@github.com:yulduzetta/budget-tracker.git
$ cd budget-tracker
$ npm install
```
  
## Usage 
```typescript
npm start
```
